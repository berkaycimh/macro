⚡ PROJECT OVERVIEW
Bu makro programı AutoHotkey (AHK) altyapısı kullanılarak, maksimum hız ve minimum gecikme için optimize edilmiştir. 🚀

🛠️ CORE FEATURES
💎 ULTRA FAST: En düşük MS değerleri ile ışık hızında işlem.

🔒 SAFE & CLEAN: Gereksiz kaynak tüketimi yapmayan saf AHK kodu.

🌀 LOOP ENGINE: Kesintisiz döngü ve toggle (aç/kapat) desteği.

📦 ASSEMBLY & REQUIREMENTS
🛠 Engine: AutoHotkey v1.1+ or v2.0

⚙️ OS: Windows 10 / 11 (x64)

🖱 Status: Operational ✅

🚀 QUICK START
AHK Kur: Bilgisayarında AutoHotkey yüklü olduğundan emin ol.

Kodu İndir: .ahk uzantılı dosyayı indir.

Çalıştır: Sağ tıkla ve Run Script diyerek başlat! 🎯
